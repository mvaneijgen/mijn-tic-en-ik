# mijnticenik

> Nuxt.js project

## Build Setup

```bash
# install dependencies
$ yarn install # Or npm install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
